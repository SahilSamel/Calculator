Instructions

  • This is a simple calculator that can perform basic arithmatic Addition, Subtraction, Multiplication, Division.
  • An expression, as a whole, can be entered. For example:-
    6+5
  • To input negative numbers,simply input the "-" sign before the number.
  • To get result, simply press on the equal to button.
  • To backspace your input press the backspace button once.
  • Not putting any input or inputting incorrect expressions will result in an error
  • For erasing all inputs,outputs and errors, longpress on the backspace button.
  • Choose between light and dark theme, by pressing the theme button (Dark theme is default)
